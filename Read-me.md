House of Posh – Luxury Beauty & Wellness HTML Template
Author: khayacomply
Version: 1.0
Created: December 2025

ABOUT:
House of Posh is a premium, responsive HTML template designed for luxury beauty studios, wellness centers, and perfume boutiques. Built with clean HTML5, CSS3, and vanilla JavaScript — no frameworks.

FEATURES:
- Responsive layout (mobile, tablet, desktop)
- Hero slideshow with auto-rotate
- Elegant gallery with lightbox (prev/next/close)
- Perfume product showcase
- FAQ accordion
- Mobile hamburger menu
- WhatsApp-integrated booking
- Splash screen on load
- Google Fonts: Great Vibes + Poppins
- Font Awesome icons

FOLDER STRUCTURE:
- index.html, about.html, services.html, etc.
- assets/css/style.css
- assets/js/main.js
- assets/images/ (all visuals)
- documentation/documentation.html
- preview/thumbnail.png
- README.txt
- LICENSE.txt

CUSTOMIZATION:
- Edit text/content directly in HTML
- Replace images in assets/images/ (keep filenames)
- Update colors in style.css (#b890d1 for accents)
- Change WhatsApp number in all HTML files + main.js

BROWSER SUPPORT:
Chrome, Firefox, Safari, Edge (latest versions)
