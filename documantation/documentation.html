House of Posh – Documentation

Luxury Beauty, Wellness & Perfume HTML Template

1. Introduction

Thank you for purchasing House of Posh — a premium, responsive HTML template designed for luxury beauty salons, wellness studios, and fragrance boutiques in Johannesburg and beyond.

This template is built with clean, semantic HTML5, CSS3, and vanilla JavaScript — no frameworks like Bootstrap, ensuring lightweight performance and full customization freedom.

3. How to Edit

3.1. Text & Content

Open any `.html` file in a code editor (e.g., VS Code). Edit headings, paragraphs, and button text directly in the HTML.

Example:

<h2>✨ Bridal Glamour</h2>
<p>Flawless makeup for your perfect day</p>

3.2. Images

All images are in assets/images/. Replace any image by keeping the **same filename** (e.g., hero1.jpg).

Supported formats: .jpg, .png, .svg.

3.3. Colors & Fonts

Brand colors and fonts are defined in assets/css/style.css:

  • Primary color: #b890d1 (violet/pink accent)
  • Background: #000
  • Headings: Great Vibes (Google Font)
  • Body: Poppins (Google Font)

To change colors, search for #b890d1 in the CSS and replace.

3.4. WhatsApp Booking

Your WhatsApp number is used in:

  • Floating button (all pages)
  • Bookings page (dynamic link)
  • Footer social icons

To update it, edit these files:

  • All `.html` files: search for 27772449092
  • `assets/js/main.js`: update the WhatsApp link in the bookings section
